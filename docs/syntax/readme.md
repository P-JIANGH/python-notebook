# 语法简记

### 数据类型

1. 整数

2. 浮点数

3. 字符串

4. 布尔值

5. 空值

6. 变量

### 集合数据类型

1. list `list = [1, 2, 3]`

2. tuple `list = (1, 2, 3)`

3. dict `list = {a: 1, b: 2}`

4. set `list = (['a', 'b'])`

### 结构控制语句

#### 条件判断`if-elif-else`

```python
if condition1:
  ...
elif condition2:
  ...
elif condition3:
  ...
else:
  ...
```

#### 循环

`for` 循环：

```python
  for x in [1, 2, 3]:
    print(x)
```

`while` 循环：

```py
  n = 0
  while n > 5:
    n += 1
    print(n)
```

`break` 与 `continue` 功能与其他语言相同
